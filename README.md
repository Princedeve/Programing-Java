# ☕ Java Programming

This repository contains various **Java programs** and **mini projects** that cover both **Core Java** and **Advanced Java** concepts.  
It is created to strengthen programming logic, object-oriented understanding, and placement preparation skills.

---

## 📚 Topics Covered
- Basics of Java (Syntax, Data Types, Operators)  
- Conditional Statements & Loops  
- Methods and Recursion  
- Object-Oriented Programming (OOPs)  
  - Classes and Objects  
  - Inheritance  
  - Polymorphism  
  - Abstraction  
  - Encapsulation  
- Arrays and Strings  
- Exception Handling  
- File Handling  
- Collections Framework  
- Multithreading  
- Mini Projects and Practice Programs  

---

## 🛠️ Tools Used
- **Language:** Java  
- **Compiler/IDE:** VS Code, IntelliJ IDEA, Eclipse, or BlueJ  
- **JDK Version:** Java 8 or above  

---

## 🎯 Purpose
This repository is created for:
- Practicing **Core and Advanced Java**  
- Preparing for **college labs, assignments, and placements**  
- Building **logic and OOP concepts** through examples and projects
